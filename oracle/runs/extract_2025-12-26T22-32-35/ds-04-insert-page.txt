## insert_page 工具概述

insert_page工具是Google Slides API中用于创建新幻灯片的核心功能。它通过batchUpdate方法实现，需要在请求体中包含一个CreateSlideRequest对象。

## CreateSlideRequest 完整JSON结构

```json
{
  "objectId": string,  // 可选
  "insertionIndex": integer,  // 可选
  "slideLayoutReference": {  // 可选
    object(LayoutReference)
  },
  "placeholderIdMappings": [  // 可选
    {
      object(LayoutPlaceholderIdMapping)
    }
  ]
}
```

## 字段详细说明

### 1. objectId
- **类型**: string
- **是否必需**: 否
- **含义**: 用户提供的对象ID。如果指定ID，它必须在演示文稿中的所有页面和页面元素中是唯一的。
- **格式要求**: 
  - ID必须以字母数字字符或下划线开头（匹配正则表达式`[a-zA-Z0-9\_]`）
  - 剩余字符可以包括这些字符以及连字符或冒号（匹配正则表达式`[a-zA-Z0-9\_-:]`）
  - ID长度必须在5到50个字符之间（含）
  - 如果不指定ID，则会生成一个唯一的ID

### 2. insertionIndex
- **类型**: integer
- **是否必需**: 否
- **含义**: 可选的从零开始的索引，指示在哪里插入幻灯片
- **默认行为**: 如果不指定索引，幻灯片将在演示文稿末尾创建

### 3. slideLayoutReference
- **类型**: LayoutReference对象
- **是否必需**: 否
- **含义**: 要插入的幻灯片的布局引用，基于当前母版
- **布局来源**:
  - 前一个幻灯片索引的母版
  - 如果insertionIndex为零，则为第一张幻灯片的母版
  - 如果没有幻灯片，则为演示文稿中的第一个母版
- **错误处理**: 如果在当前母版中找不到LayoutReference，将返回400错误请求错误
- **默认行为**: 如果不指定布局引用，幻灯片将使用预定义的`BLANK`布局

### 4. placeholderIdMappings
- **类型**: LayoutPlaceholderIdMapping对象数组
- **是否必需**: 否
- **含义**: 从布局上的占位符到从指定布局在幻灯片上创建的占位符的对象ID映射的可选列表
- **使用条件**: 仅在指定`slideLayoutReference`时才能使用

## LayoutReference 结构

```json
{
  // 联合字段 kind 只能是以下之一:
  "predefinedLayout": enum(PredefinedLayout),  // 预定义布局
  "layoutId": string  // 演示文稿中的布局ID
}
```

### PredefinedLayout 枚举值

Google Slides API提供了多种预定义布局类型：

- `BLANK`: 空白布局
- `CAPTION_ONLY`: 仅包含说明
- `TITLE`: 仅包含标题
- `TITLE_AND_BODY`: 标题和正文
- `TITLE_AND_TWO_COLUMNS`: 标题和两列
- `TITLE_ONLY`: 仅标题
- `SECTION_HEADER`: 章节标题
- `SECTION_TITLE_AND_DESCRIPTION`: 章节标题和描述
- `ONE_COLUMN_TEXT`: 单列文本
- `MAIN_POINT`: 主要观点
- `BIG_NUMBER`: 大数字

## 使用最佳实践

1. **ID唯一性**: 确保objectId在演示文稿中唯一，建议使用UUID或特定前缀+时间戳的组合
2. **索引范围**: insertionIndex应在0到当前幻灯片总数之间
3. **布局选择**: 根据内容需求选择合适的预定义布局，减少后续需要添加的元素
4. **占位符映射**: 使用placeholderIdMappings可以更好地控制占位符，特别是在需要后续操作特定占位符时
5. **错误处理**: 处理可能返回的错误，如400错误（无效请求）和404错误（未找到演示文稿）

## 常见错误及解决方案

1. **ID冲突错误**: 确保objectId唯一，或让API自动生成ID
2. **布局不存在错误**: 验证预定义布局名称或布局ID是否正确
3. **索引越界错误**: 确保insertionIndex在有效范围内
