## Design Principles

### Visual Hierarchy
- Establish a clear hierarchy with title, subtitle, and body text
- Use size, weight, and color to guide the viewer's eye
- Most important information should be immediately visible
- Limit each slide to one main idea or concept

### Simplicity
- Follow the "less is more" approach
- Remove unnecessary elements, words, and decorations
- Use white space strategically to improve comprehension
- Avoid clutter and competing visual elements

### Consistency
- Maintain consistent formatting throughout the presentation
- Use the same color scheme, fonts, and layout patterns
- Establish predictable patterns for similar content types
- Create visual rhythm through repeated elements

### Readability
- Ensure text is legible from a distance
- Use high contrast between text and background
- Keep text lines short (6-8 words per line is ideal)
- Use appropriate font sizes (minimum 24pt for body text)

## Fonts

### Font Selection
- Sans-serif fonts are preferred for digital presentations (Arial, Helvetica, Calibri, Segoe UI)
- Limit to 2-3 fonts maximum per presentation
- Choose fonts that are available across platforms
- Consider readability on different screen sizes

### Typography Guidelines
- Titles: 36-44pt, bold weight
- Subtitles: 28-32pt, medium weight
- Body text: 24-28pt, regular weight
- Captions/notes: 18-20pt, regular weight
- Line spacing: 1.2-1.5 times font size
- Character spacing: Slightly increased for titles (1.05-1.1)

## Color Rules

### Color Palette
- Limit to 3-4 colors maximum (excluding white and black)
- Use 60-30-10 rule: 60% primary, 30% secondary, 10% accent
- Choose colors that align with brand or topic
- Ensure sufficient contrast (WCAG AA standard: 4.5:1 for normal text)

### Color Application
- Use color to highlight important information
- Avoid using color as the only way to convey information
- Consider color blindness (avoid problematic combinations like red-green)
- Use darker text on lighter backgrounds for optimal readability

### Recommended Color Schemes
- Professional: Navy blue, gray, white, with a single accent color
- Creative: Complementary or analogous color schemes
- Technical: Monochromatic with limited accent colors
- Data visualization: Distinct colors for different data series

## Layout Constraints

### Grid System
- Use a consistent grid (typically 3×3 or 4×4)
- Align elements to grid lines for visual consistency
- Maintain consistent margins and padding (typically 5-10% of slide width)
- Use the rule of thirds for focal point placement

### Content Organization
- Limit to 6-7 elements per slide maximum
- Use visual grouping to connect related elements
- Create clear paths for the eye to follow
- Balance elements across the slide (avoid clustering in one area)

### Spatial Relationships
- Use proximity to show relationships
- Maintain consistent spacing between similar elements
- Use alignment to create visual connections
- Create visual flow from top-left to bottom-right

## Shape Types

### Basic Shapes
- Rectangles and squares: For containers and text boxes
- Circles and ovals: For highlighting, diagrams, and visual interest
- Triangles: For directional cues and hierarchy
- Lines: For separation, connection, and emphasis

### Shape Guidelines
- Use shapes consistently (same style throughout)
- Limit to 2-3 different shape styles per presentation
- Apply subtle shadows or borders for depth when needed
- Use rounded corners for a modern, friendly appearance

### Advanced Elements
- Icons: Simple, universally recognizable symbols
- Arrows: For showing flow and direction
- Callouts: For emphasis and attention
- Charts: For data visualization (bar, pie, line, etc.)

## Additional Rules

### Animation
- Use sparingly and purposefully
- Keep animations simple and consistent
- Avoid distracting transitions
- Consider audience and context

### Images
- Use high-quality, relevant images
- Ensure images support rather than distract from content
- Apply consistent treatment (borders, shadows, etc.)
- Consider image placement and sizing

### Data Visualization
- Choose appropriate chart types for your data
- Label clearly and concisely
- Use color consistently for the same data categories
- Keep charts simple and focused on the key message
