## 引入Material Design Icons

要在HTML中使用Material Design Icons，首先需要在页面头部引入图标库：

```html
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
```

## 基本语法

Material Design Icons的基本使用语法非常简单：

```html
<i class="material-icons">icon_name</i>
```

其中，`icon_name`是您想要使用的图标名称，例如：`home`、`settings`、`favorite`等。

## 关键点说明

1. **引入方式**：通过Google Fonts CDN引入Material Design Icons
2. **基本语法**：使用`<i class="material-icons">icon_name</i>`来显示图标
3. **自定义样式**：
   - 通过CSS类设置图标颜色：`.icon-blue { color: #2196f3; }`
   - 通过CSS类设置图标大小：`.icon-large { font-size: 36px; }`
4. **卡片组件**：使用Flexbox布局创建响应式卡片，图标、内容和操作按钮排列整齐

## 扩展功能

1. **图标变体**：Material Design Icons还提供了轮廓图标、圆角图标等变体：
   ```html
   <!-- 轮廓图标 -->
   <i class="material-icons-outlined">favorite_border</i>
   
   <!-- 圆角图标 -->
   <i class="material-icons-round">favorite</i>
   
   <!-- 锐角图标 -->
   <i class="material-icons-sharp">favorite</i>
   ```

2. **动态图标**：可以通过JavaScript动态更改图标：
   ```javascript
   // 更改图标
   document.querySelector('.card-icon .material-icons').textContent = 'check_circle';
   ```
